# Скрипт для чистой Ubuntu Server
1. Зайти на новый/свежеустановленный сервер
2. ```sudo apt install git -y```
5. ```git clone https://github.com/idpro1313/vpnserver.git```
7. ```cd vpnserver```
8. ```sudo sh install.sh```
9. Portainer будет досутпен по адресу `https://your_server_addr:9443/`
10. Дашбоард для управления Wireguard будет доступен по адресу `http://your_server_addr:10086/`
